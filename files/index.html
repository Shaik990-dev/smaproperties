🔥 Firebase Status ⚠️ Not Connected

Firebase saves your properties online so any device can see them. Follow these steps once:

1
Go to console.firebase.google.com → Sign in with your Google account → Click "Add project" → Name it sma-builders → Create project
2
Inside your project → Click the </> Web icon → App nickname: SMA Website → Register App → Copy the firebaseConfig object shown
3
Go to Build → Realtime Database → Create database → Start in TEST MODE → Enable. Copy the database URL (e.g. https://sma-builders-default-rtdb.firebaseio.com)
4
Open index.html in Notepad → Find PASTE_YOUR_apiKey_HERE → Replace all values with your copied config → Save the file
5
Re-upload to Vercel → Firebase will connect automatically ✅

🚀 Deploy on Vercel (Step by Step)

1
Download your project files — You need 2 files: index.html and vercel.json. Keep them in one folder named sma-project.
2
Go to vercel.com → Click "Sign Up" → Choose "Continue with Google" → Free account.
3
Click "Add New Project" → Choose "Upload" → Drag and drop your sma-project folder → Click Deploy. Your site goes live in 30 seconds!
4
You'll get a free link like sma-project.vercel.app — share this with anyone to test.
5
Connect your GoDaddy domain → In Vercel: Project Settings → Domains → Add smaproperties.com → Vercel shows you DNS values to add in GoDaddy (see table below).

🌐 Connect GoDaddy Domain to Vercel

In GoDaddy → My Products → DNS → Add these records:

TypeNameValueTTL
A@76.76.21.21600
CNAMEwwwcname.vercel-dns.com600

⏱️ DNS changes take 10 minutes to 24 hours to work. After that, smaproperties.com will open your website!
